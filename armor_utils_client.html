apply_buffered_changes()[source]¶

Apply buffered manipulations to the ontology when its reference has been initialized in buffered manipulation mode

Returns:

True if ontology is consistent, False otherwise
Return type:

bool
Raises:
  • armor_api.exceptions.ArmorServiceCallError – if call to ARMOR fails
  • armor_api.exceptions.ArmorServiceInternalError – if ARMOR reports an internal error

Note

It returns the boolean consistency state of the ontology. This value is not updated to the last operation if you are working in buffered reasoner or manipulation mode!
